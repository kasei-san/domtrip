# 誤答ログ（間隔反復用）

間違えた問題を**論点単位**で記録する。問題そのものでなく「論点」を覚えるのが目的。
ミス種別で打ち手が変わる: 知識不足→暗記 / 読み違い→設問を二度読む / 計算ミス→手順テンプレ化 / 選択肢比較不足→消去法の練習。

復習タイミング: 翌日 → 3日後 → 1週間後 → 2週間後（正解できたら間隔を伸ばす。間違えたら翌日に戻す）。
